## LLM Skills Repository

A structured collection of reusable LLM skills, prompts, and evaluation methods.

This repository is focused on building practical, testable capabilities rather than storing raw prompts.

### What is a “Skill”?

A skill is a complete unit that includes:
- A clearly defined task (e.g. generate a presentation)
- A prompt that performs the task
- Example inputs and outputs
- (Optional) an evaluation method

## How to Use
1. Pick a skill
2. Review the prompt.md
3. Run it with your own input
4. Compare with examples
5. Iterate or extend

## Goals of This Repo

- Build reusable LLM capabilities
- Improve prompt design through iteration
- Develop evaluation methods (LLM-as-a-judge)
- Create a personal playbook for applied AI
-  Work in Progress
